## Local testing

   From the root directory, run the following command:
        python -m http.server

   The page should now be available at `http:localhost:8000`


## Technologies used

    * HTML5
    * CSS Flexbox modul
    * Python for a local HTTP server
    * GitHub Pages for Deployment
